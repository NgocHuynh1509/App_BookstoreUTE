# Coupons Management (Admin)

This module provides the Admin Coupons Management UI and data access.

## Scope
- List/search/filter coupons
- Create/update/delete coupons
- Show quick stats (active, expiring soon, total used, top used)

## Run (manual)
- Open app and go to Tab "Khác" -> "Mã giảm giá".
- Use the "+ Thêm mã" button to create or edit coupons.
